🚀 Proyecto: dispositivo electrónico con Arduino

Diseña, monta y programa un dispositivo que sense → think → act: lee al menos un sensor y actúa con al menos un actuador.

1La idea

Vas a construir un dispositivo funcional con Arduino que combine al menos:

  • 1 sensor (LDR, NTC, ultrasonido, IR, pulsador, potenciómetro…)
  • 1 actuador (LED, zumbador, servo, motor CC, relé…)
  • Una regla lógica programada: «si pasa X, hago Y».

Trabajaréis por parejas en 4-5 sesiones de aula-taller. La entrega final: dispositivo funcional + código + memoria PDF + defensa oral con demo.

2Ideas de partida (elige una o propón)

💡 Farola automática

LDR + LED. Se enciende automáticamente al anochecer.

🚗 Sensor de aparcamiento

Ultrasonidos + zumbador. Pita más rápido cuanto más cerca.

🚦 Semáforo con paso de peatones

3 LEDs + pulsador. Cuando un peatón pulsa, el semáforo cambia.

🌡️ Alarma de temperatura

NTC + LED rojo + zumbador. Si supera un umbral, alerta.

🤖 Mini-robot evita-obstáculos

Ultrasonidos + 2 motores + chasis. Avanza y gira al detectar pared.

🚪 Detector de visitas

Sensor IR + zumbador + LED. Avisa cuando entra alguien por la puerta.

🎵 Mini-piano

5 pulsadores + zumbador. Cada tecla suena una nota distinta.

🌱 Maceta inteligente

Sensor humedad + LED + bomba/relé. Alerta cuando hay que regar (o riega solo).

🔧 Tu idea

Habla con el profesor para validar la dificultad.

3Fases del proyecto

🎯 1. Definir

Problema real + qué hace el dispositivo + para quién.

📐 2. Esquema

Esquema eléctrico con todos los componentes y conexiones.

🔌 3. Montaje

Protoboard, cables, sensores y actuadores conectados.

💻 4. Programa

Pseudocódigo → código Arduino. Subir y probar.

🧪 5. Pruebas

Funciona en distintas condiciones reales.

🔁 6. Iterar

Detectar fallos y mejorar al menos UNA cosa.

📦 7. Carcasa

Cartón pluma, caja impresa 3D, MDF cortado a láser.

🎤 8. Defender

5 min con demo + memoria PDF + vídeo.

4Entregables

  1. Dispositivo funcional: montado, programado y funcionando.
  2. Memoria técnica (PDF, 6-8 págs.): problema, esquema eléctrico, lista de componentes con justificación, código comentado, pruebas realizadas, mejora aplicada.
  3. Código fuente (.ino) bien comentado.
  4. Vídeo (60-90 s) mostrando el dispositivo en uso.
  5. Defensa oral (5 min) con demo en directo.

✅ Material necesario por pareja:
· 1 Arduino UNO + cable USB
· 1 protoboard + cables jumper
· Resistencias (220 Ω, 1 kΩ, 10 kΩ)
· 2-3 LEDs
· Pulsadores
· Sensores según el proyecto (LDR, NTC, HC-SR04...)
· Actuadores según proyecto (zumbador, servo, motor CC + driver)
· Pilas o cargador para alimentación autónoma
⚠️ Seguridad: Nunca conectar el Arduino a 230 V directamente (usar siempre relé con aislamiento). Cuidado con la polaridad de LEDs y motores (se pueden quemar). Si algo huele a quemado, desconecta inmediatamente.
