Actividades · U05 Circuitos y robótica

14 actividades en 5 bloques. Trabaja la ley de Ohm, los circuitos serie/paralelo/mixtos, la electrónica básica y la programación de Arduino. Cada una con solución colapsable.

Bloque 1 · Ley de Ohm

1Ohm directo

Una bombilla de R = 25 Ω se conecta a una pila de 5 V. Calcula la corriente.

$I = V/R = 5/25 = \boldsymbol{0{,}2\ \text{A}} = 200\ \text{mA}$.

2Ohm con despeje

Por una resistencia desconocida pasa 0,4 A cuando se conecta a 12 V. ¿Cuál es su valor?

$R = V/I = 12/0{,}4 = \boldsymbol{30\ \Omega}$.

Bloque 2 · Serie y paralelo

3Serie

Tres resistencias R₁ = 10 Ω, R₂ = 20 Ω y R₃ = 30 Ω están en serie con una pila de 12 V. Calcula R_eq, I total y V en R₂.

$R_{eq} = 10 + 20 + 30 = \boldsymbol{60\ \Omega}$.
$I = 12/60 = \boldsymbol{0{,}2\ \text{A}}$ (igual en todas).
$V_2 = I \cdot R_2 = 0{,}2 \cdot 20 = \boldsymbol{4\ \text{V}}$.

4Paralelo

R₁ = 6 Ω y R₂ = 3 Ω en paralelo, alimentadas por 9 V. Calcula R_eq, las corrientes parciales y la total.

$R_{eq} = \dfrac{6 \cdot 3}{6+3} = \dfrac{18}{9} = \boldsymbol{2\ \Omega}$.
$I_1 = 9/6 = \boldsymbol{1{,}5\ \text{A}}$ · $I_2 = 9/3 = \boldsymbol{3\ \text{A}}$.
$I_{total} = I_1 + I_2 = \boldsymbol{4{,}5\ \text{A}}$. Comprobación: $9/2 = 4{,}5$ A ✓

Bloque 3 · Circuitos mixtos

5Mixto básico (serie con paralelo dentro)

Pila 12 V. R₁ = 2 Ω en serie con un paralelo formado por R₂ = 8 Ω y R₃ = 8 Ω. Calcula R_eq, I total y V en R₂.

Paralelo: $R_p = \dfrac{8 \cdot 8}{8+8} = 4\ \Omega$.
Serie: $R_{eq} = 2 + 4 = \boldsymbol{6\ \Omega}$.
$I_{total} = 12/6 = \boldsymbol{2\ \text{A}}$.
$V_1 = 2 \cdot 2 = 4$ V. $V_p = 12 - 4 = \boldsymbol{8\ \text{V}}$ (= $V_2 = V_3$).

6Mixto inverso (paralelo con serie dentro)

Pila 10 V. Una rama: R₁ = 3 Ω + R₂ = 7 Ω en serie. En paralelo con esa rama, R₃ = 5 Ω. Calcula R_eq, I total e I que circula por R₁.

Serie rama 1: $R_{12} = 3+7 = 10\ \Omega$.
Paralelo con R₃: $R_{eq} = \dfrac{10 \cdot 5}{15} = \boldsymbol{3{,}33\ \Omega}$.
$I_{total} = 10/3{,}33 = \boldsymbol{3\ \text{A}}$.
Tensión en cada rama: 10 V. Por rama 1+2: $I_1 = 10/10 = \boldsymbol{1\ \text{A}}$. Por R₃: $I_3 = 10/5 = 2$ A. Suma = 3 A ✓

7Mixto con 4 resistencias

Pila 24 V. R₁ = 4 Ω en serie con (R₂ = 6 Ω || R₃ = 6 Ω || R₄ = 6 Ω). Calcula R_eq, I total y la corriente por cada paralela.

3 resistencias iguales en paralelo: $R_p = 6/3 = 2\ \Omega$ (truco: N resistencias iguales R en paralelo = R/N).
Serie: $R_{eq} = 4 + 2 = \boldsymbol{6\ \Omega}$.
$I_{total} = 24/6 = \boldsymbol{4\ \text{A}}$.
$V_1 = 16$ V. $V_p = 8$ V. Por cada paralela: $I = 8/6 \approx \boldsymbol{1{,}33\ \text{A}}$. Comprueba: 1,33 × 3 ≈ 4 A ✓

8Mixto con potencia

Mismo circuito que la actividad 5 (12 V, R₁ = 2 Ω en serie con (R₂ = 8 || R₃ = 8)). Calcula la potencia disipada en R₁ y la total.

Del ej. 5: $I = 2$ A, $V_1 = 4$ V.
$P_1 = V_1 \cdot I = 4 \cdot 2 = \boldsymbol{8\ \text{W}}$. Comprueba con $I^2 R = 4 \cdot 2 = 8$ W ✓
Potencia total: $P_{total} = V \cdot I_{total} = 12 \cdot 2 = \boldsymbol{24\ \text{W}}$.

10Resistencia limitadora para un LED

Quieres conectar un LED rojo (V_f = 2 V, I_f = 20 mA) a una pila de 9 V. ¿Qué resistencia limitadora necesitas? Elige el valor comercial más cercano por encima entre: 220 Ω, 330 Ω, 470 Ω, 1 kΩ.

La resistencia «consume» la diferencia de tensión: $V_R = 9 - 2 = 7$ V.
$R = V_R / I_f = 7 / 0{,}02 = \boldsymbol{350\ \Omega}$.
Valor comercial por encima: 470 Ω (para que circule un poco menos de los 20 mA y el LED no fuerce).

Bloque 5 · Arduino y robótica

12Lee y predice

¿Qué hace este programa de Arduino?
int led = 9; void setup() { pinMode(led, OUTPUT); } void loop() { for (int i = 0; i <= 255; i++) { analogWrite(led, i); delay(10); } for (int i = 255; i >= 0; i--) { analogWrite(led, i); delay(10); } }

El LED hace un efecto de respiración (fade in / fade out): se enciende gradualmente desde apagado hasta brillo máximo (255) en 2,5 s, y se apaga gradualmente hasta cero en otros 2,5 s. Se repite indefinidamente. Usa PWM con analogWrite en el pin 9 (los pines PWM en Arduino UNO son 3, 5, 6, 9, 10 y 11).

13Encuentra el error

El programa debería encender un LED cuando se pulsa un botón. No funciona: el LED está siempre apagado. ¿Cuál es el error?
int led = 13; int boton = 2; void setup() { pinMode(led, INPUT); pinMode(boton, OUTPUT); } void loop() { if (digitalRead(boton) == HIGH) { digitalWrite(led, HIGH); } else { digitalWrite(led, LOW); } }

Los pinModes están al revés: el led está como INPUT (entrada, no puede generar señal) y el botón como OUTPUT (salida, no puede leer). Lo correcto:
pinMode(led, OUTPUT);
pinMode(boton, INPUT_PULLUP);
Y la condición debe ser == LOW (con INPUT_PULLUP, el pin está en HIGH y baja a LOW al pulsar).

14Diseña el programa

Diseña un programa Arduino para un parking inteligente: si un sensor de ultrasonidos detecta que se acerca un coche a menos de 30 cm, se enciende un LED rojo y suena un zumbador. Si está más lejos, todo apagado. LED en pin 9, zumbador en pin 8, trig en 12, echo en 11.
int led = 9; int buzz = 8; int trig = 12; int echo = 11; void setup() { pinMode(led, OUTPUT); pinMode(buzz, OUTPUT); pinMode(trig, OUTPUT); pinMode(echo, INPUT); } long medirDistancia() { digitalWrite(trig, LOW); delayMicroseconds(2); digitalWrite(trig, HIGH); delayMicroseconds(10); digitalWrite(trig, LOW); long dur = pulseIn(echo, HIGH); return dur * 0.0343 / 2; // cm } void loop() { long d = medirDistancia(); if (d < 30) { digitalWrite(led, HIGH); tone(buzz, 1000); } else { digitalWrite(led, LOW); noTone(buzz); } delay(100); }

Comentarios: el sensor HC-SR04 mide la duración del eco; se convierte a cm con la velocidad del sonido (0,0343 cm/μs / 2 porque va y vuelve). La función tone() emite una onda cuadrada a la frecuencia indicada.
